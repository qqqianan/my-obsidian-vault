# Skills Hub

## 定义

Skills Hub 是团队共享 Skills 的集中入口。它承担类似技能库、插件市场或团队知识目录的角色，使成员可以发布、查找、安装和管理 Skills。

## 在团队中的作用

Skills Hub 的关键作用是把分散能力集中呈现。成员不需要知道某个流程最初由谁写在本地，也不需要从聊天记录里翻找配置，只要在 Hub 中找到对应 Skill，就可以复用既有能力。

## 应包含的信息

- Skill 名称和用途
- 适用场景与不适用场景
- 安装或启用方式
- 维护人或责任团队
- 版本与更新时间
- 权限要求和数据边界
